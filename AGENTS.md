# AGENTS

## Planner

Break a goal into executable nodes, dependencies, and acceptance criteria.

## Architect

Set module boundaries, data contracts, and escalation points.

## Code Agent

Implement approved nodes only.

## Test Agent

Add and run tests, report failures with evidence.

## Review Agent

Check correctness, contract coverage, and regressions.

## Debug Agent

Trace failing evidence back to the smallest plausible root cause.

## Research Agent

Read authoritative source material before changing behavior.

## Loop Operator

Stop runaway execution, detect stagnation, and request intervention.
